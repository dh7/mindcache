Examples

Real-world use cases for AI agents

// LLM Tool Integration
import { mindcache } from 'mindcache';

// Set up memory with tags
mindcache.set_value('userName', 'Alice', { tags: ['user'] });
mindcache.set_value('favoriteColor', 'blue', { tags: ['preferences'] });

// Auto-generate LLM tools for AI agents
const tools = mindcache.get_tools();

// AI can now call write_userName() to update memory
// Tools are automatically generated for all writable keys

// Generate system prompt with all context
const systemPrompt = mindcache.get_system_prompt();
// "userName: Alice. You can rewrite userName by using..."
// "favoriteColor: blue. You can rewrite favoriteColor by..."

Auto Tool Generation

Automatically generate LLM tool schemas so AI agents can read and write memory using function calls.

// Template Injection with Recursion
import { mindcache } from 'mindcache';

// Set up nested templates
mindcache.set_value('firstName', 'Alice', { template: true });
mindcache.set_value('greeting', 'Hello {firstName}!', { template: true });
mindcache.set_value('message', '{greeting} Welcome to our app.');

// Recursive template resolution
const result = mindcache.inject_stm('{message}');
// "Hello Alice! Welcome to our app."

// Use temporal keys
const prompt = mindcache.inject_stm(
  'Today is {$date} at {$time}. User: {firstName}'
);
// "Today is 2025-10-03 at 14:30:00. User: Alice"

Recursive Template Injection

Build complex prompts with nested templates and automatic temporal context injection.

// Reactive Updates & Tag Search
import { mindcache } from 'mindcache';

// Subscribe to specific key changes
mindcache.subscribe('userName', (newValue) => {
  console.log(`Name changed to: ${newValue}`);
});

// Subscribe to all changes
mindcache.subscribe_to_all(() => {
  console.log('Memory state updated');
});

// Organize with tags
mindcache.set_value('age', 25, { tags: ['user', 'demographics'] });
mindcache.set_value('city', 'NYC', { tags: ['user', 'location'] });

// Search by tag
const userKeys = mindcache.get_keys_by_tag('user');
// ['age', 'city']
